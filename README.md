## 🚀 Project Overview
In this analysis, I performed end-to-end Exploratory Data Analysis (EDA) to uncover why customers leave. Key technical steps included:
- **Data Cleaning:** Replacing white-spaces and casting `TotalCharges` to float.
- **Mapping:** Converting binary flags to categorical labels for better visualization.
- **Visualization:** Creating a multivariate grid to compare 9 different service features against churn.
